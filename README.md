### Goal: Become comfortable with Python and logical thinking.

### Week 1 Topics:

 - Variables

 - Loops
 - Functions
 - Lists (start)

### Practice:

 - 50+ small problems
 - Prime numbers
 - Multiplication tables
 - Pattern problems

 - Convert all programs into functions

### Week 2 Topics:

 - Nested lists

 - Remove duplicates
 - Reverse list
 - Frequency counter

Mini project:

  * Find average
  * Highest/lowest
  * Grade distribution

### Week 3 Topics:

 - Dictionaries
 - Dictionary operations
 - Sets
 - String manipulation
 - Practice problems:
 - Word frequency counter
 - Contact book

### Week 4 Topics:

 - File handling
 - Exception handling
 - Modules
 - Code organization

Mini project:
 - Expense Tracker (CLI)

  * Add expense
